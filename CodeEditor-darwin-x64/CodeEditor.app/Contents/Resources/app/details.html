Introduction

Notre multi-platforme Editeur de Code permet aux developeurs d'ecrire leur code html, css et JavaScript d'une façon simple.

Dans notre projet on a utiliser des nouveaux technologies pour la creation de softwares comme le Framework Electron.js, Node.js, Bootstrap 4, jQuery, Sweet Alert 2, CodeMirror et Split.js. aussi il est multi-platforme (Windows, Mac, Linux),

Coloriser Le Code

Editer le code (Html, Css et JavaScript) avec une belle colorisation. vous pouvez aussi editer les autres types de texte comme (.txt, .md ..etc).

Parcourir le dossier

Parcourir dans les dossier et les fichiers de votre projet façilement, avec l'arbre du projet.

Operation sur les fichiers

Vous pouvez creer, ouvrir et sauvgarder un fichier de tout les types des textes (.html, .java, .txt, ...etc) dans un seul temps. Vous pouvez aussi ouvrir un dossier qui porte plusieurs fichier

Démarrer

Dans le lancement de l'éditeur, la page d'accueil apparaît, vous pouvez lire à propos de l'éditeur dans la page des détails.

Pour ouvrir votre premier projet. Clickez sur le menu Fichier (Électron pour Mac) ensuite ouvrir un dossier. votre dossier. ensuite la page va actualiser pour mettre votre dossier en place. finalement vous pouvez ouvrir les fichiers du projet

Manipuler les fichiers

L'éditeur contient un système des fenêtres pour faciliter l'édition de plusieurs fichiers au même temps. La fermeture d'une fenêtre peut se faire en clichant sur le button "X" ou bien dans Le menu Fichier > Fermer La fenêtre. dans tous les cas si votre fichier n'est pas enregistrer un message vont apparaitre pour la confirmation de la fermeture.

Quand vous fermez l'éditeur complètement s'il y a un seul fichier n'est pas enregistré un message de confirmation apparaitre. par contre dans MAC si vous utilisez CMD+Q (Tuer le processuce) même s'il y a un fichier n'est pas enregistrer l'éditeur va être fermé sans confirmation (ALT+F4 : dans le cas du Windows).

Changer la taille du police

Vous pouvez changer la taille de la police utiliser dans notre éditeur comme vous voulez. Juste clickez sur vue > Incrementer La Police (pour incrémenter la taille), Decrementer La Police (pour Decrementer la taille) ou bien Remise La Police (pour remise la taille par défaut)

Operation sur les Fichier dans L'arbre

Dans l'arbre des fichiers d'un certain dossier vous pouvez faires 2 opérations
-Copier la direction du fichier: qui va copier l'emplacement du fichier dans votre ordinateur
-Supprimer un fichier: qui va supprimer le fichier complètement. cetter opération va actualiser l'éditeur et le relancer

Operation sur le text dans l'editeur

Dans l'éditeur vous pouvez faire plusieurs opérations sur le texte (le code). il y a 4 opérations:
-Sélectionner tous: permettre de sélectionner tout le texte dans le fichier
-Copier: Copier un certain texte sélectionnait. s'il y a aucun texte sélectionné l'éditeur va prendre toute la line comme défaut
-Couper: Couper un certain texte sélectionnait. s'il y a aucun texte sélectionné l'éditeur va prendre toute la line comme défaut
-Coller: Coller un texte déjà copier ou couper.

Technologies Utuliser

Electron.js

Électron est un framework pour créer des applications natives avec les techniques web comme Javascript, HTML et CSS avec tous les web frameworks comme (Angular, React js, vue js, ... etc.)

Node.js

Node js est un moteur d'exécution Javascript basé sur le moteur Javascript V8 de Chrome. Le paquet de l'écosystème de Node js, npm, est le plus grand écosystème de bibliothèques open source dans le monde.

ES6 (ECMAScript 6)

ECMAScript est une spécification de langage de script standardisée par ECMAScript International. Il est utilisé par les applications pour activer les scripts côté client.

CodeMirror

CodeMirror est un éditeur de texte implémenté en JavaScript pour les navigateurs. Il est spécialisé pour l'édition de code

Bootstrap

Bootstrap est le framework le plus populaire pour la conception réactive des sites web , il permettra aussi des creer des belles designe

JQuery

JQuery est une bibliothèque riche de Javascript, permet de minimiser le code Javascript. elle est utilisée dans le côté client (Front-End)

Split.Js

Split js est un framework qui permet de diviser la fenêtre aux plusieurs parties avec une fonctionnalité de glisser les fenêtres

Sweet Alert 2

Sweet Alert 2 est un module qui permettre des créer des fenêtres de dialog, et des messages d'alert, d'error et d'information...etc., il est utilisé pour Électron. Js
